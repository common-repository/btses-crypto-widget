= From your WordPress dashboard =
1. Visit "Plugins > Add New"
2. Search for "BTSE's Crypto Widget"
3. Activate "BTSE's Crypto Widget" from your Plugins page.
4. Add widgets on yourdomain.com/wp-admin/widgets.php page.

= From WordPress.org =
1. Download "BTSE's Crypto Widget".
2. Upload the "BTSE's Crypto Widget" directory to your "/wp-content/plugins/" directory, using your favorite method (ftp, sftp, scp, etc...).
3. Activate "BTSE's Crypto Widget" from your Plugins page.
4. Add widgets on yourdomain.com/wp-admin/widgets.php page.
